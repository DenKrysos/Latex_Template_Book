% \sfdefault - Sans Serif Font
% \ttdefault - TypeWriter Font

% pnc 	New Century Schoolbook
% ppl 	Palatino
% ptm 	Times

% Standartmäßig verfügbare Schriftarten:
% cmss 	Computer Modern Sans Serif (default)
% lmss 	Latin Modern Sans Serif
